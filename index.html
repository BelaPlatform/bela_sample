The Bela Pattern Library

Welcome to the Bela pattern reference.

The purpose of a pattern library is to provide a cohesive description of the design elements (colours, font choices, appearance, etc) for an application or brand. This library is for the Bela browser-based IDE and web properties and describes all current design rules, but extends into all areas of the Bela brand (print and virtual). It is active and updated as it evolves.

In the forkable version of this reference the CSS is exhaustively componentized - for the sake of clarity and reproducability. This library uses jQuery, and all functionality is contained in the global.js file (under 100 lines). You're invited to fork this repo and have a look around.

For suggestions, please open a Github issue.

Last updated: Dec 2016, by Astrid Bin.

Print and other fonts

The fonts used for the IDE are Poppins and Space. However, for print the main font is Montserrat. It's slighty more decorative which looks better on paper and as a standalone element (ie, as a URL in a logo or a single word on a webpage), but it was busy for the IDE. A note on text over colours

When making text over a colour lighter, do not make the text grey - always use black (RGB 0,0,0 or #000000) and adjust the alpha value.

For example, the box on the left uses

color: rgba(0,0,0,0.5)

,

while the one on the right uses

color: #7d7d7d

.

The difference in readability is significant.
